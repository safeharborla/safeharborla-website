## Project Overview

This is the website for **Safe Harbor LA**, a grant-funded violence prevention and trauma recovery program at Harbor-UCLA Medical Center and the Lundquist Institute. The site should reflect the program's mission: heal trauma, uplift community, and promote positive change through mentorship and leadership development.

No code exists yet — this repository contains only design assets and content. The task is to build the website from scratch.

## Design Guidance

- The brand uses blue and yellow (see `Safe Harbor Blue.png` / `Safe Harbor Yellow.png`)
- The tone should be warm, community-focused, and professional — appropriate for a trauma-informed healthcare program
- The audience includes potential clients/patients, community partners, donors, and referring clinicians
